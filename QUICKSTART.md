# Quick Start Guide

Get your Portfolio Maker up and running in 5 minutes!

## 🚀 Installation (2 minutes)

```bash
# Clone the repository
git clone https://github.com/johnny-ops/portfolio-maker.git
cd portfolio-maker

# Install dependencies
npm install

# Start the app
npm start
```

The app opens at `http://localhost:3000`

## ✏️ Create Your Portfolio (3 minutes)

### Step 1: Personal Info (1 min)
1. Click "Personal Info" tab
2. Enter your name and title
3. Upload a profile picture
4. Write a short bio
5. Add your email and social links

### Step 2: Add Skills (30 seconds)
1. Click "Skills" tab
2. Click "Add Skill"
3. Enter 3-5 of your top skills
4. Set proficiency levels

### Step 3: Add Projects (1 min)
1. Click "Projects" tab
2. Click "Add Project"
3. Add 2-3 of your best projects
4. Include descriptions and links

### Step 4: Choose Template (30 seconds)
1. Use the template dropdown in header
2. Try all 8 templates
3. Pick your favorite

### Step 5: Export (30 seconds)
1. Click "Export" dropdown
2. Choose "Export as PDF" or "Export as HTML"
3. Done! Your portfolio is ready
