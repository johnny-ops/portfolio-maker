## Responsive Design

### Desktop View (1920x1080)
- Side-by-side form and preview
- Full feature access
- Optimal editing experience

### Tablet View (768x1024)
- Stacked layout
- Touch-optimized controls
- Swipe between form and preview

### Mobile View (375x667)
- Single column layout
- Bottom navigation
- Mobile-optimized inputs

## Dark Mode

All templates and the interface support dark mode:
- Automatic theme detection
- Manual toggle
- Consistent across all views
- Reduced eye strain
