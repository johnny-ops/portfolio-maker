# 🚀 Features

## Core Features

### 📝 Multi-Section Form Builder
Create your portfolio by filling out intuitive forms across multiple sections:

- **Personal Information**
  - Full name, professional title, and profile picture
  - Bio/About me section
  - Contact details (email, phone, location)
  - Social media links (GitHub, LinkedIn, Twitter, Portfolio)

- **Skills Management**
  - Add unlimited skills
  - Set proficiency levels (Beginner, Intermediate, Advanced, Expert)
  - Dynamic add/remove functionality
  - Visual skill display in templates

- **Projects Showcase**
  - Multiple project entries
  - Project images with upload preview
  - Project descriptions and technologies used
  - GitHub repository links
  - Live demo links

- **Work Experience**
  - Job title and company information
  - Employment duration
  - Detailed role descriptions
  - Timeline visualization

- **Education History**
  - School/University name
  - Degree information
  - Graduation year

### 🎨 8 Professional Templates

Choose from 8 beautifully designed templates:

1. **Minimal** - Clean and professional
2. **Dark Mode** - Modern dark theme with purple accents
3. **Creative** - Colorful gradient design
4. **Glassmorphism** - Modern frosted glass effect
5. **Japanese Minimal** - Zen-inspired simplicity
6. **Bold Typography** - Statement-making design
7. **Card Grid** - Pinterest-style layout
8. **Timeline** - Story-driven narrative

### 🎨 Theme Customization

- **Color Picker** - Choose your brand colors
- **Preset Color Schemes** - Quick theme selection
- **Live Preview** - See changes in real-time
- **Dark/Light Mode** - Toggle between themes

- Real-time updates as you type
