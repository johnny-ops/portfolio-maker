# Next Features - Portfolio Maker Roadmap

This document outlines upcoming features and enhancements planned for the Portfolio Maker application.

## 🚀 High Priority Features

### 1. Firebase Integration
**Status**: Planned  
**Description**: Cloud storage for portfolio data
- User authentication (Google, Email/Password)
- Save portfolios to cloud database
- Access portfolios from any device
- Share portfolio links with unique URLs
- Portfolio versioning and history

### 2. Drag-and-Drop Section Reordering
**Status**: Planned  
**Description**: Reorder portfolio sections with drag-and-drop
- Drag to reorder skills, projects, experiences
- Visual feedback during dragging
- Save custom section order
- Reset to default order option

### 3. Custom Theme Color Picker
**Status**: Planned  
**Description**: Personalize portfolio colors
- Color picker for primary/secondary colors
- Live preview of color changes
- Preset color schemes
- Save custom color palettes
- Apply colors across all templates

### 4. Additional Portfolio Templates
**Status**: In Progress  
**Description**: Expand template library
- **Template 4**: Modern Glassmorphism design
- **Template 5**: Minimalist Japanese-inspired layout
- **Template 6**: Bold Typography-focused design
- **Template 7**: Card-based Grid layout
- **Template 8**: Timeline-based narrative style

### 5. Advanced Export Options
**Status**: Planned  
**Description**: Enhanced export capabilities
- Export as React component
- Export as Vue component
- Generate deployment-ready package
- Export with custom domain setup
- Batch export (PDF + HTML + JSON)
